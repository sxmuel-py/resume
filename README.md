## ⚡ Overivew

This project is a modern, responsive portfolio designed to reflect a "security-first" development philosophy. It avoids unnecessary complexity in favor of crisp typography, smooth transitions, and a robust dark mode implementation.

### Key Features
- **Minimalist Aesthetic**: Inspired by Apple-grade product design with a focus on hierarchy and whitespace.
- **Dynamic Dark Mode**: Persistent theme switching with smooth CSS transitions.
- **Typewriter Animation**: Creative hero section highlighting core technical competencies.
- **Professional Iconography**: Standardized Lucide-React icons for a polished tech feel.
- **Fully Responsive**: "Reshaped" mobile layout for optimal viewing on any device.
- **Secure by Design**: Clean code structure with no external dependencies for core UI elements.

## 🛠️ Tech Stack

- **Core**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) + [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS (Custom tokens & variables)
- **Deployment**: [Vercel](https://vercel.com) / [Netlify](https://www.netlify.com)

## 🏗️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sxmuel-py/resume.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
